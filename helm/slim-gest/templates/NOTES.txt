IMPORTANT CONFIGURATION NOTES:
================================
Before deploying, please ensure:

1. **GPU Support**: Your Kubernetes cluster must have GPU support enabled
   - NVIDIA device plugin installed
   - GPU nodes properly labeled

2. **Models Volume**: Update values.yaml with the correct path to your models:
   volumes.models.hostPath.path: /your/path/to/models
   
   The models directory should contain:
   - nemotron-ocr-v1/checkpoints/

3. **Build the Docker Image**: 
   docker build -t slim-gest:latest .
   
   If using a registry:
   docker tag slim-gest:latest your-registry/slim-gest:latest
   docker push your-registry/slim-gest:latest
   
   Then update values.yaml:
   image.repository: your-registry/slim-gest

4. **Resource Requirements**: The default configuration requests:
   - 1 GPU
   - 4Gi memory (request) / 8Gi (limit)
   - 2 CPU cores
   
   Adjust these in values.yaml based on your workload.
